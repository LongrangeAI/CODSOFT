## ğŸ“‚ Dataset Description

The dataset used in this project is the **Iris Dataset**, which contains 150 samples from three species of Iris flowers:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

Each sample contains the following features:

* `sepal length (cm)`
* `sepal width (cm)`
* `petal length (cm)`
* `petal width (cm)`
* `species` (target label)

You can find this dataset [here](https://archive.ics.uci.edu/ml/datasets/iris) or load it via a CSV file named `IRIS.csv`.
